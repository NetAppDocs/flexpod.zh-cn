== 简介

NetApp 和 Cisco 合作提供 Cisco Intersight ，这是 FlexPod 生态系统的单一窗格视图。这种简化的集成为 FlexPod 基础架构和 FlexPod 解决方案中的所有组件创建了一个统一的管理平台。您可以通过 Cisco Intersight 监控 NetApp 存储， Cisco 计算和 VMware 清单。它还允许您编排或自动化工作流，以便同时完成存储和虚拟化任务。
