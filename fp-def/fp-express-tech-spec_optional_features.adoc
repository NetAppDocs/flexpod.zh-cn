== iSCSI 启动选项

FlexPod 快速架构使用 iSCSI 启动。iSCSI 启动选项的最低要求包括：

* 在 NetApp 存储控制器上激活的 iSCSI 许可证 / 功能
* NetApp 存储控制器 HA 对中每个节点上的一个双端口 10Gbps 以太网适配器
* Cisco UCS 服务器中支持 iSCSI 启动的适配器
